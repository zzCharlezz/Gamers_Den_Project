INSCRIPTION RELOADED - PyGame to Web Game
==========================================

IMPORTANT REQUIREMENT:
DAPAT NASA ISA TALAGA MAIN CLASS ANG BUONG LARO PRA MACONVERT ANG PYGAME SA PYGBAG

INSTALLATION:
Open CommandPrompt and type:
	pip install pygbag

USAGE:
Use cd command to change directory to your pygame folder:
	cd path\to\your\pygame\folder
	pygbag your_game.py

After running pygbag, it will provide a local URL (usually http://localhost:8000) 
to play your game in the browser.
